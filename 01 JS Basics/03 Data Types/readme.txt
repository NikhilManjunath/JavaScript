In JS, variable can take many types:

1. Numerical Values 
  eg: var width = window.innerWidth; var pi = 3.14;
     
2. String - to create a string, you have to put the value in quotes
  eg: var name = "Nikhil"
     
3. Boolean: eg: var status = false;

4. Object: var topic = document.getElementById('myId')
     
     
5. Array: var links = getElementsByTagName('a); -. gets all elements with a tag 
  
  Accessing array elements is done using index
     var links = getElementsByTagName('a);
     document.write(links[0]); -> returns the first link in the array
